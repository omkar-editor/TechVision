## 🔐 Form Validation Rules

### Name Field
- **Required**: Yes
- **Min Length**: 2 characters
- **Pattern**: Letters and spaces only
- **Error Messages**: 3 types (required, too short, invalid format)

### Email Field
- **Required**: Yes
- **Pattern**: Valid email format (name@domain.com)
- **Real-time Validation**: On blur and input

- **Required**: No (optional)
- **Min Length**: 10 digits (if provided)

- **Type**: Dropdown selection
- **Options**: General, Services, Support, Partnership, Other

### Message Field

- **Min Length**: 10 characters
- **Error Message**: Clear feedback

- **Compliance**: Privacy agreement

### 1. Quick Start (30 seconds)
```bash
# Navigate to project folder
cd D:\VS code\Battelground\Vortex

# Open in browser (Windows)
# Double-click index.html
# OR open with Start > index.html
```

### 2. Local Server (Better for testing)
```bash
# Start Python server
python -m http.server 8000

# Open browser
# http://localhost:8000
```

- Toggle device toolbar (Ctrl+Shift+M)
- Select different device sizes
